# Dnt

Search to find current local time and time zone

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'dnt'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dnt

## Usage

    $ dnt search 'san jose'
